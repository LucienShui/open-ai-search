# Open AI Search

> This project is still under heavy development.

Open AI Search, inspired by perplexity and metaso.

## Usage

```shell
cp config.example.yaml config.yaml
pip install -r requirements.txt
python3 main.py
```

Then open `http://localhost:8000` by default.
